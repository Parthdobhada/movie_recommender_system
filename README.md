# 🎬 Movie Recommender System

A content-based Movie Recommender System developed using Machine Learning and Streamlit.  
The application recommends movies similar to the selected movie using cosine similarity and displays movie posters, IMDb ratings, genres, release years, and similarity scores using the OMDb API.

---

## 🚀 Features

- Movie recommendations using Content-Based Filtering
- Cosine Similarity algorithm
- Movie posters using OMDb API
- IMDb ratings display
- Genre and release year information
- Similarity percentage score
- Interactive Streamlit user interface
- Netflix-style recommendation layout

## ▶️ How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/Parthdobhada/movie_recommender_system.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Streamlit App

```bash
streamlit run app.py
```

## 🧠 Machine Learning Concept Used

The system uses **Content-Based Filtering** with **Cosine Similarity** to recommend movies based on movie metadata similarity.
